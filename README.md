## 👨‍💻 About me

I started as a **Full-Stack Developer** (Java / Spring Boot, REST APIs) and moved deliberately into **Cybersecurity & DevSecOps** — so I don't just secure software, I understand how it's built and shipped.

- 🎓 Computer Engineering graduate, finishing an **MSc in Applied Cybersecurity for Industrial Environments**
- 🔐 Cybersecurity intern @ **Accenture** — Zero Trust & SASE architectures
- ⚙️ I work where **development, cloud and security meet**: GitOps, Kubernetes, policy-as-code and detection engineering
- 🏴 Active on **Hack The Box** & **TryHackMe**, plus hackathons
- 🗣️ English · Spanish · Catalan · German (learning)

## 🚀 Featured Projects

### 🛡️ Secure DevSecOps Platform on Kubernetes — *MSc Thesis*
A GitOps-driven, security-hardened delivery pipeline built end to end.
- **Shift-left security** in CI with automated image & dependency scanning (**Trivy**)
- **GitOps continuous delivery** with **ArgoCD** + **GitHub Actions**
- **Policy-as-code** admission control via **Kyverno**
- **Network segmentation** with **Calico** network policies on **Kubernetes**

> Demonstrates supply-chain security, automated guardrails and least-privilege networking — the core of a modern DevSecOps workflow.

### 🔎 Home SOC — Detection, Response & Threat Intel Lab
A self-hosted security operations stack, built and wired together by hand.
- **SIEM / XDR** monitoring with **Wazuh**
- **Threat intelligence** with **MISP**
- **Intrusion prevention** with **CrowdSec** + **Fail2ban**
- **WAF & edge** with **Traefik** + **Coraza**
- **SOAR-style automation** with **n8n**

> Detection engineering, threat-intel enrichment and automated response — the blue-team counterpart to the platform above.
